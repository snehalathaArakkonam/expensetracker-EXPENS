# 💰 EXPENS – Smart Expense Tracker

A modern, responsive, and intelligent Expense Tracker web application designed to help users manage income, expenses, budgeting, and financial insights in a clean and interactive way.

🌐 Live Demo:
- 🔗 Netlify: https://expensetracker-expens.netlify.app/
- 🔗 Vercel: https://expensetracker-one-zeta.vercel.app/

---

## ✨ Features

### 🧾 Core Features
- Add, edit, and delete income & expenses
- Categorize transactions (Food, Travel, Rent, etc.)
- Date-wise tracking system
- Real-time balance calculation (Income – Expense)
- Simple and clean dashboard view

---

### 📊 Smart Analytics
- Interactive charts (Pie & Bar graphs)
- Category-wise expense breakdown
- Monthly & weekly analysis
- Spending insights (top spending categories)

---

### 🎯 Budget & Savings System
- Set monthly budget limits
- Smart alerts when exceeding budget
- Savings goal tracker with progress bar
- Spending behavior insights

---

### 🔁 Advanced Features
- Recurring expenses (Rent, EMI, Subscriptions)
- Calendar-based expense tracking
- Expense search & filtering system
- Export data as CSV/PDF
- No-spend day streak tracking
- Income vs Expense comparison insights

---

### 👤 User System
- User profile setup (Name + Role selection)
- Supports Students, Employees, Workers, Business users & custom roles
- Personalized dashboard based on user type
- Greeting system (Hello, [User Name])

---

### 🎨 UI/UX Design
- Glassmorphism design with modern UI
- Bento grid layout structure
- Dark & Light mode support
- Fully responsive (Mobile + Desktop)
- Smooth animations & micro-interactions
- Skeleton loading screens
- Modern form elements & styled inputs

---

## 🛠️ Tech Stack

- HTML5 – Structure
- CSS3 – Glassmorphism UI & Responsive Design
- JavaScript (ES6) – Logic & Functionality
- Chart.js – Data Visualization
- LocalStorage – Data Persistence

---

## 📱 Design Philosophy

EXPENS is built with a **user-first financial mindset**, focusing on:

- Financial awareness
- Smart spending habits
- Easy budgeting
- Visual financial insights
- Clean and distraction-free UI
