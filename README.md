##  OVERVIEW

EXPENS is a modern expense tracking web application designed to help users manage income, expenses, budgeting, and financial insights in a simple and efficient way.

Built for students, employees, workers, freelancers, and household users.

---

##  FEATURES

###  Core Features
- Add, edit, delete income & expenses  
- Category-based tracking  
- Real-time balance calculation  
- Date-wise transaction tracking  
- Clean dashboard UI  

---
## 🧩 PROBLEM SOLVED

Many people struggle to:
- Track daily expenses  
- Control overspending  
- Maintain savings discipline  

💡 EXPENS solves this by providing:
- Clear financial tracking  
- Budget alerts  
- Visual insights  
- Simple and easy UI
